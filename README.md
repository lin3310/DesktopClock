# DesktopClock

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**桌面時鐘** 是一個以 C# WPF / .NET 8.0 開發的極簡桌面時鐘應用，設計以低功耗與簡潔介面為主。

## 作者
- 林楷庭 / kaiting LIn (GitHub: lin3310)

## 為甚麼我要做這個?

甚麼叫做我只是要個時鐘還要Rainmeter/桌布引擎？

## 建置
需要 .NET 8.0：

```bash
dotnet build
dotnet run --project DesktopClock.csproj
```
